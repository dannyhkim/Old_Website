PROJECTS

Shoot 'Em Up

Programming Language: Java

A shooter genre game in which 3 different iterative waves of enemies fire bullets at you, the player, who must dodge them all using the keyboard controls (WASD).

As the player, you are also able to shoot back at the enemies using the "v" key. Player bullets are differentiated through the colour green.

The player scores 1 point every time they survive a wave of enemies or when they kill an enemy by hitting them with 3 bullets. The player starts the game with 5 lives, and the game ends when all lives are lost.

Map Creator/Maze Game

Programming Language: Java

A visual Map Creator in which users may build their own map with the side tools that consist of blocks, inventory, and a player.

Users who have finished creating their map can save it as a .txt file.

To play the game, users must now open the Game Panel (a different window) and use the LOAD function to open their saved map.

To start the maze game, users can press PLAY and will be able to move their heart player with the arrow keys. The player must first retrieve money or a key to enter through the ATM machines blocking the paths.

An end game window pops up once the player has finished going through the maze.
